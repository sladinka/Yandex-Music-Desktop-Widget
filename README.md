# Yandex-Music-Desktop-Widget
Yandex Music Desktop Widget Стильный, минималистичный виджет для рабочего стола Windows, который отображает текущий трек, обложку альбома и живой эквалайзер для Yandex Music. Полностью адаптируется под цвет обложки в реальном времени.
## Инструкция по установке

1. **Клонируйте репозиторий:**
   ```bash
   git clone [https://github.com/your-login/Yandex-Music-Desktop-Widget.git](https://github.com/your-login/Yandex-Music-Desktop-Widget.git)
   cd название-репозитория
2. **Установите необходимые зависимости:**
   `pip install customtkinter pystray winsdk pillow soundcard numpy`
3. **Запустите приложение:**
   `python3 main.py`
